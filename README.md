# Парсер вакансий с HeadHunter

Программа для поиска, анализа и сохранения вакансий с платформы HeadHunter (hh.ru).

## Возможности

- Поиск вакансий по ключевому слову
- Сохранение результатов в JSON-файл
- Сортировка вакансий по уровню зарплаты
- Фильтрация по ключевым словам в описании
- Автоматическая обработка ошибок API
- Логирование операций

## Технологии

- Python 3.10+
- Requests (для работы с API)
- Pytest (для тестирования)
- JSON (формат хранения данных)
- Logging (система логирования)
- ООП (классы, наследование, инкапсуляция)

## Установка и запуск

1. Клонируйте репозиторий:
```
git clone https://github.com/ваш-username/hh-parser.git
cd hh-parser
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Запустите программу:
```
python main.py
```

## Пример использования
Введите поисковый запрос: Python разработчик
Введите количество вакансий (макс. 100): 50

Введите количество вакансий для топа: 10

Топ вакансий:
Senior Python Developer | Зарплата: 250000-300000 RUR
Python Backend Engineer | Зарплата: 200000-250000 RUR

Введите ключевое слово для поиска: Django
